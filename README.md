# Autolink token server
A spring boot mongodb rest api server with vuejs frontend 
## Prerequisite
```
1. java 17
2. Node v16.13.1
3. NPM 8.19.2
4. Apache Maven 3.5.3
5. Running mongodb server on port 27017
```

## Project setup
```
mvn clean install
```

### run the server
```
mvn spring-boot:run
```

### server url
```
http://localhost:8989/
```
